# Changelog

История пользовательских релизов Backpacker. Формат версий: `super-release.major-epic.minor-feature.patch`.

## 1.1.2.8 - 2026-07-02

- Добавлена iPhone PWA readiness: manifest получил `id`, standalone-настройки проверены, добавлена iOS touch icon 180x180 и iOS meta tags.
- Добавлен компактный onboarding `Установить Backpacker` для iPhone в браузерном режиме.
- Onboarding не показывается в установленном standalone-приложении и не возвращается после закрытия в рамках текущей версии.
- В подсказке зафиксировано ограничение iOS: Home Screen web app использует отдельный `localStorage`, поэтому приложение лучше установить до создания первой поездки. Автоматический перенос данных из Safari не выполняется.

## 1.1.2.7 - 2026-07-02

- Добавлен первый шаг живого шаринга поездки через Supabase: автор может открыть доступ по ссылке, скопировать ссылку, обновлять опубликованную поездку после сохранений и отозвать доступ.
- Публичная ссылка открывает поездку в read-only режиме без авторизации гостя.
- Настройка `Показывать смету` управляет серверной проекцией: при выключении финансовые поля не отдаются из Edge Function.
- Личные нерасшаренные поездки остаются только в `localStorage`; в Supabase отправляется только опубликованный snapshot конкретной поездки.
- Управление ссылкой привязано к текущей anonymous-сессии автора. После потери anonymous-сессии управление существующей ссылкой не восстанавливается до появления постоянных аккаунтов.
- Raw share-token хранится только локально для повторного копирования; в Supabase хранится только hash.
- UI шаринга уточнен: деталка ссылки открывается под кнопкой `Открыть доступ по ссылке`, повторный клик скрывает ее, кнопка копирования приведена к хромовому стилю.

## 1.1.2.6 - 2026-07-01

- Зафиксирован канонический вид PDF-экспорта поездки.
- PDF повторяет композицию мобильной шапки поездки, показывает бюджетные карточки и раскладывает карточки поездки сеткой.
- Действия PDF разделены на `Скачать` и `Поделиться`.

## 1.1.2.0

- В `Поделиться поездкой` добавлен PDF-сценарий с настройками сметы, заметок и элементов без даты.
- PDF строится локально из данных поездки и отправляется через системный file share или скачивается fallback-ом.

## 1.1.1.0

- Добавлено копирование карточки из редактора в другой день текущей поездки, в `Без даты` или в другую поездку.

## 1.1.0.1

- Подготовлен закрытый donation flow без публичной активации.

## 1.1.0.0 - 2026-06-28

- Зафиксирована первая стабильная продуктовая линия Backpacker.
- Канонизированы карточки, дорожки дней, drag/drop, бюджет, экспорт и нижняя навигация.
