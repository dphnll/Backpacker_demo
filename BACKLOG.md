# Backpacker Backlog

Живой список ближайших продуктовых доработок. Перед началом новой итерации быстро просматривать этот файл и переносить актуальные пункты в план работы.

## Shared Trips

### Near-term

- На карточке приглашенной поездки показывать долю гостя, когда у него есть accepted participant link/allocation. До этого явно подписывать общий бюджет как `Бюджет поездки`, чтобы не путать viewer-сценарий с личной долей.
- Дать гостю редактировать идею до отправки. Для уже отправленной pending-идеи MVP-путь остается `отозвать и создать заново`, пока не появится безопасный edit-flow.
- На карточке авторской групповой поездки показывать яркий входящий маркер предложений, например торчащий сверху ярлычок `+1 идея` или `Есть предложения · N`. Нужен легкий backend-count, чтобы главная не загружала все предложения по всем поездкам.

### Later

- Обычная авторизация и восстановление доступа без потери текущих `auth.uid()` связей.
- Более явная модель ролей для гостя: viewer / participant.
- Multi-device sync для личных поездок после появления постоянных аккаунтов.
